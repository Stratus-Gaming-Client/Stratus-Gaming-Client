Plans

Automatic Game Installation

This will allow games to be installed just by submitting a form to the app. You may have seen a file called install.php, this is the start of the automatic game installation. However, it is still in development and doesn't currently work.

No Game Installation

This will come long after automatic installation. This will allow games to be loaded from the Stratus servers every time you want to play. This has been the plan since the development of Stratus first started.
